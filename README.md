# Parametric Insurance Mean-Variance Analysis
This project compares a traditiona indemnity insurance contract with a parametric insurance contract using a Mean-Variance utility framework, with an additional stress test for a fat-tailed severity distributions (Pareto).

# Prerequisites
Ensure you have Python 3.10+ installed on you machine

# Setup and Execution
```bash
cd Parametric_Insurance_Mean_Variace
pip install -r requirements.txt
python main.py
```
